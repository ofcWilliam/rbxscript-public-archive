# rbxscript-public-archive

[![Repository Size](https://img.shields.io/github/languages/code-size/ofcWilliam/rbxscript-public-archive)](https://github.com/ofcWilliam/rbxscript-public-archive)

A public preservation archive dedicated to backing up Roblox scripts, UI libraries, and community resources. The goal of this repository is to ensure that essential development tools remain accessible even if original repositories or hosting sites are taken down.


## 🗄️ Archive Policy & Standards

To ensure resources remain usable after being moved, this archive employs a specialized storage standard:

* **Functional Optimization**: I modify the primary `Source.lua` files to update asset links, icon paths, and internal dependencies. This ensures that UI libraries and scripts continue to load correctly with loadstrings from this new location.
* **Integrity Preservation**: I always include an **Untouched Source** version and a **Date-Labeled .zip** of the original repository. This allows you as a developer to see the original code before any of my fixing modifications was applied.
* **No Malicious Tampering**: Modifications are strictly limited to path-fixing and link redirection. No backdoors, trackers, or hidden logic are ever added.
